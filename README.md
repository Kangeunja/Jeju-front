# 🏝 Jeju Travel Portal
> **제주도 관광지, 축제, 역사 정보를 소개하는 관광 포털 웹 서비스 시스템**

> 본 프로젝트는 기존 제주 관광 포털 사이트를 벤치마킹하여 퍼블리싱을 진행하고 React 기반으로 인터페이스를 구현했습니다.
> 또한 MSW(Mock Service Worker)를 활용하여 API 호출 환경을 구축했습니다.

🔗 Deploy
https://jeju-portfolio.netlify.app/

## 🛠 기술 스택 및 개발 환경
### **Frontend**
- React
- TypeScript
- Axios
- Swiper

### **Mock API**
- MSW (Mock Service Worker)

### Design & Tool
- Figma
- Github

## ✨ 주요 기능
- 제주 관광지, 축제, 역사 정보 콘텐츠 제공
- Swiper를 활용한 제주 대표 음식 슬라이드 UI 구현
- useState 기반 상태 관리
- MSW 기반 API 요청 구조 구현
